# PixelVault

PixelVault is a secure personal cloud workspace that allows users to store files, manage photos, access temporary email, and organize digital content in one place.

The application integrates with Google services such as Gmail to allow users to securely access and manage their email inside the PixelVault interface.

## Features

- Secure file storage
- Photo gallery management
- Temporary email tools
- Notes and chat workspace
- Personal cloud tools
